# Receiptiq 🧾

Upload a grocery receipt and instantly see expiry dates 
for your items plus meal ideas based on what's expiring soon.

## Features
- 📸 Receipt scanning via OCR
- 📅 AI-powered expiry date estimation
- 🍳 Meal suggestions based on your groceries
- 💰 Automatic receipt total extraction

## Tech Stack
- Python / FastAPI (backend)
- OCR.space API (text extraction)
- OpenAI GPT-4o-mini (expiry + meal logic)
- HTML/CSS/JS (frontend)
